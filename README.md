# PortBlaster
Simple Port Scanner in Rust

build in rust

``cargo build --release``

``cargo run --release -- -H 1.1.1.1 -s 1 -e 100 -T 500 `` --> 1.1.1.1 or one.one.one.one or google.com.......

in your cmd or terminal Windows Linux Mac

--> ``portblaster -h``
